# Comparison of Claude-Derived Table vs. Manual CSV (Baron 1983)

## Purpose
This folder contains scripts and outputs used to evaluate the reliability of a table generated by Claude against a manually transcribed version (`Baron_1983.csv`) of Baron et al. (1983) Table 1.

The goal is to determine whether discrepancies arise from:
- errors in the Claude-derived extraction, or
- transcription errors in the manually created CSV.

---

## Summary of Results

The comparison identified **2 mismatches** in species names:

mismatches[,7]

# A tibble: 2 × 1
  species_csv_baron1983
  <chr>                
1 Oryzoricles talpoides
2 Avahi I. occidentalis

Both mismatches can be explained as manual transcription errors in Baron_1983.csv, rather than issues with the Claude-derived table.

Conclusions
Based on this audit, all entries in the Claude-derived table fall into one of three categories:
Exact matches
Discrepancies due to manual transcription errors
Entries present only in the Claude-derived table
